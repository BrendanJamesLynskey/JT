# JT

Interactive HTML projects for JT.

## Rotor — Low Flight

A VR-style cockpit helicopter game. Fly low along a winding road through procedurally
generated terrain, dodging pylons, arches, cables, balloons and buildings. Options
for difficulty, time of day and weather. Touch controls on iPhone (dual virtual
sticks), or Desktop mode with mouse-look + keyboard on laptops and PCs —
auto-detected for your device.

[Play Rotor — Low Flight](https://brendanjameslynskey.github.io/JT/helicopter-vr.html)

## Zombie Apocalypse

A browser-based first-person zombie shooter game.

[Play Zombie Apocalypse](https://brendanjameslynskey.github.io/JT/zombie-fps.html)

## Marketing Secrets Behind Roblox Bundles

An interactive presentation for kids exploring the marketing techniques used in Roblox.

[View the Roblox Marketing Presentation](https://brendanjameslynskey.github.io/JT/marketing_techniques.html)
